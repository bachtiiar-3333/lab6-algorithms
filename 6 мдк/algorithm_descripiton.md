# Описание алгоритма сортировки пузырьком

## Входные данные:
- Неотсортированный массив чисел

## Выходные данные:
- Отсортированный по возрастанию массив

## Принцип работы:
1. Многократный проход по массиву
2. Сравнение соседних элементов
3. Обмен элементов при нарушении порядка сортировки
4. Уменьшение области прохода после каждого цикла

## Сложность алгоритма:
- Временная сложность: O(n²)
- Пространственная сложность: O(1)
